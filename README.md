# NBA & G-League Shot Quality Analysis

A shot quality and selection analysis using real NBA and G-League shot location data, built to separate shot-making ability from shot selection.

## Project Overview
This project builds a custom Expected Field Goal Percentage model based on shot location, then compares it to actual shooting performance to answer a key player development question: is a player's efficiency driven by good decisions, good talent, or both?

- **Section 1** — Shot value by zone (NBA)
- **Section 2** — Expected vs Actual FG% model (NBA)
- **Section 3** — Shot Selection Score rankings (NBA)
- **Section 4** — Overperformers vs Underperformers (NBA)
- **Section 5** — Applying the model to G-League shot data
- **Section 6** — Case study: Nick Smith Jr., including custom shot heat maps and shot charts built in R

## Key Finding
Mid-range shooting is the biggest efficiency gap between the NBA and G-League. Nick Smith Jr.'s 2025-26 G-League shot profile shows shot-making ahead of shot quality by location.
